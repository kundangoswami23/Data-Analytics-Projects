# 🛒 Blinkit Sales Analysis

## 📌 Project Overview

This project analyzes Blinkit grocery sales data using Python and Pandas to discover valuable business insights.

---

## 🎯 Objectives

- Analyze total sales
- Find top-selling categories
- Compare city-wise sales
- Analyze monthly sales trends
- Evaluate customer ratings

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
